# mac-os

macOS development environment sandbox code.

## Purpose

Java learning exercises related to macOS development environment setup. Works
alongside the `mac-os/docs/` guides at the project root.

## Package Structure

```text
Main.java                    Entry point
development/
└── MacEnvChecker.java       Environment validation utility
```

## Dependencies

None — pure Java 21+.

## Build

```bash
./gradlew :modules:mac-os:build
```
