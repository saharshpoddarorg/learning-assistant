# BLI-019: Build Tally-Vyapaar accounting application

## Description

Build a personal accounting and business management application inspired by
Tally and Vyapaar — popular Indian accounting software. The app should handle
basic bookkeeping, invoices, expense tracking, and financial reporting for
personal or small-business use. This is a standalone personal software project.
