# notes/ — Quick Captures

> Lightweight plain-text entries. The simplest thing in the backlog.
>
> **When to use:** You have a thought, observation, or reminder that doesn't
> need priority, status, or acceptance criteria. Just write it down.
>
> **ID prefix:** `NOTE-NNN`
>
> **Promotion:** If a note becomes important, promote it to an item or idea.
