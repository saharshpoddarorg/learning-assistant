> from a different angle — status, project, or priority.
>

## View Maintenance

Views are **derived from BOARD.md** — they must be updated whenever BOARD.md changes.
When an item's status, priority, or project assignment changes, update the relevant view(s).

The AI assistant should update all affected views as part of any backlog status change.
