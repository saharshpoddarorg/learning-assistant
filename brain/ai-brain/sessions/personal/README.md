# personal/ — Personal Projects & Learning Sessions

> Captured AI chat sessions related to personal projects, self-directed learning,
> side projects, personal finance, hobbies, and non-work exploration.

---

## Categories

| Category | Folder | Use for |
|---|---|---|
| `learning` | `learning/` | Concept deep-dives, tutorials, skill-building, interview prep |
| `project-dev` | `project-dev/` | Personal project development, side-project architecture |
| `requirements` | `requirements/` | User stories, acceptance criteria, feature scoping, BDD for personal projects |
| `financial` | `financial/` | Budgeting, investment analysis, tax strategies, financial planning |
| `research` | `research/` | Personal interest research, tool evaluation, comparison analysis |
| `general` | `general/` | Sessions not fitting the above categories |

---

## Naming

```text
YYYY-MM-DD_HH-MMtt_<category>_<subject>[_v<N>].md
```

Example: `2026-03-20_05-00pm_learning_java-virtual-threads-deep-dive.md`

---

## Routing Heuristic

If the session involves personal projects, self-learning, personal finance, or topics
you'd do on your own time — it belongs here. Work-related tasks go in `sessions/work/`.
