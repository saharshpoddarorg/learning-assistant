## Naming

Files inside `deep-dive/` do **not** carry the `code-review_` category prefix — the folder path implies it.

```text
YYYY-MM-DD_HH-MMtt_<class-or-flow>-<method-or-aspect>.md
```

**Example:** `2026-05-02_04-00pm_task-manager-crud-service.md`
