## Naming

Files inside `deep-dive/` do **not** carry the `code-analysis_` category prefix — the folder path implies it.

```text
YYYY-MM-DD_HH-MMtt_<class-or-flow>-<method-or-aspect>.md
```

**Example:** `2026-05-02_03-21pm_order-service-calculate-total.md`
